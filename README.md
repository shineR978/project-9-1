# project-9-1

製作一個課程專案。學生可以尋找想要的課程與講師，講師可以新建課程並得知有幾個學生報名課程。

# install node_modules

---

### 請在 server 與 client 資料夾內 下命令 npm install

### 請確保 server/package 下有以下的套件

---

<ol>
<li>express</li>
<li>dotenv</li>
<li>bcrypt</li>
<li>mongoose</li>
<li>axios</li>
<li>joi</li>
<li>jsonwebtoken</li>
<li>passport jwt</li>
<li>passport-local</li>
<li>cors</li>
</ol>

### client/package

---

<ol>
<li>react-router-dom</li>
</ol>

---

# 啟動 client 與 server

---

## client >>> npm start

## server >>> nodemon .\index.js
